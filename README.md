# Mini Jira Workspace

A full-stack project management application built with React, Node.js, Express, and PostgreSQL.

## Features
- **Authentication**: Secure JWT-based login and registration with password hashing.
- **Kanban Board**: Real-time task management with physics-based drag-and-drop.
- **Full CRUD**: Create, Read, Update, and Delete issues directly from the dashboard.
- **Optimistic UI**: Instant state updates using React Query.

## Tech Stack
- **Frontend**: React, Tailwind CSS, @hello-pangea/dnd, React Query
- **Backend**: Node.js, Express, Prisma ORM, PostgreSQL
